For loops in python

A list comprehension is a clear and concise way in which iterable data structures can be created. Simply, comprehensions consist of brackets, either '[...]' or '{...}', containing an expression with a for clause, followed by either zero or more for and if/else statements. The type of brackets used determines the final data structure.
Ultimately, '[...]' produces a list comprehension where the data structure will be a list of simple data types. for example, ['a', 'b', 'c'] would be an example output. The '{...}' would produce a list of dicts, or key values pairs.
A list comprehension is an alternative of creating a list with a for loop.
